# Project Brief: POXY World Ecosystem (Obsidian Protocol)

## 1. Vision & Executive Summary
POXY World is an ultra-premium, high-fidelity Web3 gaming and digital collectible ecosystem. It is designed through the lens of a tactical operating system—the **"Obsidian Protocol"**—blending luxury aesthetics (Mayfair Elite) with a high-octane "Cyber-Glass" interface. The platform serves as a unified terminal for asset trading, verification, social interaction, and ecosystem governance.

## 2. Design System: "Obsidian Cyber-Glow"
*   **Visual Philosophy:** High-impact kineticism, tactical data density, and luxury minimalism.
*   **Color Palette:** 
    *   Primary Surface: #050506 (Deep Obsidian)
    *   Secondary Surface: #0d0d0f (Tactical Charcoal)
    *   Accent: #F472B6 (Muted Neon Pink / Magenta)
    *   Highlight: #F3C66F (Liquid Gold)
    *   Text: #FFFFFF (High-contrast white) and #9CA3AF (Matte Silver subtext)
*   **Typography:** HANKEN GROTESK (Tight-tracked, uppercase for headings; monospace for tactical data).
*   **Key Effects:** Glassmorphism (20px backdrop-blur), razor-thin borders (white/5 opacity), and "breathing" ambient glow transitions.

## 3. Core Modules & Functionality

### 3.1. Unified Terminal Header (The Frame)
The persistent global navigation layer.
*   **Left:** White "POXY WORLD" wordmark.
*   **Center:** Centered tabs (News, Patches, Terminal, Market, Verify).
*   **Right:** Utility matrix (Notifications, Settings, Profile Badge) and a stealth-style "Connect Wallet" button.

### 3.2. News Terminal (The Intel Feed)
The primary entry point for ecosystem updates.
*   **Bento-Grid Layout:** Large featured "Major Update" modules paired with smaller economy and hotfix cards.
*   **Interaction:** Micro-interactions on hover with a 1.02x scale and a 200ms cubic-bezier transition.

### 3.3. Tactical Collection Archive (Inventory)
A high-density asset management interface.
*   **Asset Cards:** Rarity-coded cards with glow indicators (Common, Uncommon, Rare, Epic, Legendary, Mythic).
*   **Mechanics:** "Mass Select" mode, "Burn for PC" utility, and "Sync Patches" state management.
*   **Filtering:** Tactical command-line style filters for Rarity and Type.

### 3.4. Global Ranks (Personnel Database)
A classified leaderboard of the elite echelon.
*   **Podium:** Top 3 players encased in exclusive obsidian glass modules with [RANK: 01] monograms.
*   **Data Table:** High-density list view of the Top 100 players with monospace stat readouts.
*   **Personal HUD:** A "Standing Overview" card providing real-time rank telemetry.

### 3.5. Asset Exchange (The Market)
A high-fidelity Web3 peer-to-peer marketplace.
*   **Market Pulse:** Real-time analytics block height-aligned with featured listings.
*   **Tactical Filter Bar:** Integrated search and categorical dropdowns within a unified charcoal surface.
*   **Listing HUD:** "Acquire Asset" primary command with thin-line icon secondary actions.

### 3.6. Verification Playground (Protocol Integrity)
A step-by-step cryptographic proof engine.
*   **Validation Path:** A linear, animated phase-based progress tracker (Commit -> Entropy -> Reveal).
*   **Proof Receipt:** A generated JSON/Visual receipt module for verified ledger events.
*   **Sys_log:** A real-time terminal output feed at the bottom of the viewport.

### 3.7. POXY CLUB (The Elite Lounge)
A private, gated social and governance layer.
*   **Founders Vault:** Exclusive VIP box opening mechanics with gold-shimmer animations.
*   **DAO Council:** A "Silk Edition" neomorphic voting terminal for protocol governance.
*   **Live Feed:** Real-time global chat with "Marine Glass" frosted message bubbles.

## 4. Technical Constraints
*   **Device Type:** Primary focus on Desktop (Widescreen optimized).
*   **Performance:** UI interactions must maintain a snappy 200ms duration using `cubic-bezier(0.4, 0, 0.2, 1)`.
*   **Security:** All authentication flows must utilize the "Syndicate Access Terminal" architecture.
